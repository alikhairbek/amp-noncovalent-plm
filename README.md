# Antimicrobial Peptides × Non-Covalent Interactions × Protein Language Models

Code and data for the paper *"Protein Language Models Encode the Order-Dependent Non-Covalent Determinants of Antimicrobial Activity: Interpretable Probing and Mechanism-Aware Peptide Design."*

The full analysis is a **single script** (`amino.py`) — or the same steps in a notebook (`amino.ipynb`). It runs end to end: collects peptide data from UniProt, builds physicochemical and non-covalent features, trains classifiers, probes ESM-2 embeddings, does a homology-aware evaluation, and designs new peptides.

## How to run

**Option 1 — Google Colab (recommended, free GPU):**
1. Open `amino.ipynb` in Colab.
2. `Runtime → Change runtime type → T4 GPU`.
3. `Runtime → Run all`.

**Option 2 — Locally:**
```bash
pip install -r requirements.txt
python amino.py
```

That's it. The script installs `transformers` and `cd-hit` automatically. A GPU is recommended (the ESM-2 step is slow on CPU). `cd-hit` needs Linux (e.g., Colab or Ubuntu).

## What it produces
- A cleaned, balanced dataset of peptides (`amp_dataset.csv`).
- Classification results, ESM-2 probing results, and the figures used in the paper.
- A set of designed candidate antimicrobial peptides.
